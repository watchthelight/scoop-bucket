# Scoop Bucket for HypergraphGo

This is a [Scoop](https://scoop.sh/) bucket for [HypergraphGo](https://github.com/watchthelight/HypergraphGo).

## Installation

```powershell
# Add this bucket
scoop bucket add hottgo https://github.com/watchthelight/scoop-bucket

# Install hg
scoop install hg
```

## Update

```powershell
scoop update hg
```
